Academics

Course description with Outline and Electives offered, Course Structure


CORE SUBJECTS YEAR WISE

Note : L-T-P-C system (C=L+T+P/2) per semester, that is, the number of contact hours for Lectures (L), Tutorials (T) and Practical (P) in a week.
Mathematical, Algorithmic, and Computational Thinking


1. Mathematical, Algorithmic, and Computational Thinking :- This is an introductory course on computational thinking. This course aims to motivate students with mathematical, analytical, logical thinking with emphasis on algorithmic and computational thinking. In this course we approach real-world problems in varied areas applying data analysis and computational and mathematical thinking. In this course you will learn computer science, software, algorithms, applications, and mathematics as an integrated whole.


Outline :-
  • Understanding mathematical structures
  • Logic and reasoning in mathematics
  • Mathematical notation and symbols
  • Problem-solving strategies in mathematics
  • Introduction to algorithms
  • Algorithm design and analysis
  • Flowcharts and pseudocode
  • Algorithmic problem-solving techniques

➔ Structure :- The total credit of this course is 4. Including 3-1-0-4.



Computer Organization and Programming


2. Computer Organization and Programming :- Computer Organization is the way in which a system has to be structured and it is the operational units and the interconnections between them that achieve the architectural specifications. It is the realizations of the abstract model, and it deals with how to implement the system. Computer programming helps professionals write instructions for a machine to generate desired results for a varying complexity of queries.


Outline :-
  • Overview of computer architecture
  • Von Neumann architecture
  • Binary, decimal, hexadecimal numbering system
  • Assembly language programming
  • Introduction to C language

➔ Structure :- The total credit of this course is 3. Including 3-0-0-3.



Computer Organization and Programming Lab


3. Computer Organization and Programming Lab :- This course provides an introduction to C language (high-level language) and also introduces Assembly language (low-level language).


Outline :-
  • Solve basic program in C language
  • Practical of Assembly language program

Discrete Mathematics


4. Discrete Mathematics :- Discrete mathematics is the study of mathematical structures that are countable or otherwise distinct and separable. Discrete mathematical structures include objects with distinct values like graphs, integers, logic-based statements, etc. In this course, we have covered all the topics of Discrete Mathematics for computer science like set theory, recurrence relation, graph theory, counting problem, Number theory etc.


Outline :-
  • Fundamental Principles (like set theory, functions, relations, etc.)
  • Logical reasoning (like propositional logic, quantifiers, logical equivalence, etc.)
  • Number Theory
  • Combinatorics
  • Set, Functions and Relations
  • Boolean Algebra
  • Recursion
  • Graph Theory

➔ Structure :- The total credit of this course is 4. Including 3-1-0-4.

It can perform logical operations, data retrieval or storing and data transformation by analyzing logical circuit design.

Object Oriented Programming

Data Structures and Algorithms


3. Data Structures and Algorithms :- Data Structures and Algorithms is made from two separate words- Data Structures and second is Algorithms. Data Structure is defined as a particular way of storing and organizing data in our devices to use the data efficiently and effectively and Algorithm is a step-by-step procedure or a set of rules followed to solve a specific problem or perform a particular task. It's a well-defined sequence of instructions designed to take an input, process it through a series of computational steps, and produce the desired output. We will see how to use Data Structures nicely in various problems and also see how to make efficient algorithms for given problems. And see some optimized algorithms like sorting, searching, etc…


Outline :-
  • Introduction to Algorithms
  • Recursion and its applications
  • Time and space complexity analysis
  • Sorting and Searching Algorithms
  • Introduction to Data Structures (like array, linked list, stack, queue, tree, graph, etc…)

➔ Structure :- The total credit of this course is 4. Including 3-0-2-4.



Linear Algebra


4. Linear Algebra :- Linear algebra can be defined as a branch of mathematics that deals with the study of linear functions in vector spaces. When information related to linear functions is presented in an organized form then it results in a matrix. Thus, linear algebra is concerned with vector spaces, vectors, linear functions, the system of linear equations, and matrices.


Outline :-

  • Systems of Linear Equations
  • Vector Spaces and Subspaces
  • Linear Transformations
  • Eigenvalues and Eigenvectors
  • Inner Product Spaces and Orthogonality

➔ Structure :- The total credit of this course is 4. Including 3-1-0-4.

Operating Systems


2. Operating Systems :- The course explains the evolution of computer operating systems from a simple program loader to a complete resource manager of a computing platform. We look at the structure of a modern operating system, and the design of its components. Important components like processes, memory, I/O, and file systems are studied in detail. We also study the implementation of these constructs in the context of a basic Unix-like operating systems. In the laboratory, experiments/assignments cover: system programming in Unix-like OS, multithreading, and concurrency control. At the end of the course, students are expected to have an in-depth understanding of a modern OS architecture and be able to understand and implement programs using system calls.


Outline :-
  • Introduction to Operating Systems
  • Process Management
  • Memory Management
  • File Systems
  • I/O Systems
  • Security and Protection

➔ Structure :- The total credit of this course is 4. Including 3-0-2-4.
